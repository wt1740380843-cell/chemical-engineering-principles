# DESIGN.md

## 项目与用户画像

- **产品**：化工原理学习网站，面向化工及相关专业学生、考研备考者与化工从业者
- **核心课程**：覆盖流体流动、流体输送机械、非均相物系分离、传热、蒸发、蒸馏、吸收、萃取、干燥等核心章节
- **风格定位**：温暖、精致、高级印刷质感——精装教材 + 工程笔记的数字孪生

## 气质与意象

温暖纯净的米白色像翻开一本经典教材的纸页，陶土橙的强调色如实验室的蓝焰与精密仪器的指示灯。整体传达的是：经过时间沉淀的工程智慧，而非冰冷的数字界面。

**关键词**：温暖、精致、印刷质感、陶土、纸白、工程温度

**场景意象**：翻开一本被翻旧的《化工原理》教材，页边有手写的公式批注；实验室工作台上，暖光透过锥形瓶折射出琥珀色的光；工程笔记本上，铅笔绘制的精馏塔草图旁，用橙色的荧光笔标记了关键参数。

## 字体排版

- 英文：Inter (Google Fonts) — 主字体，搭配系统原生中文字体
- 代码：Fira Code monospace — 公式与数据用
- 排版节奏：h1 厚重温暖，正文行距 1.8，长文阅读舒适
- 公式与数据用 mono 体或斜体区分

## 页面结构

### 首页
- Hero 区：品牌标志 + 课程名称 + 副标题 + CTA 按钮，温暖质感背景
- 统计条：核心数据（章节数、知识点数等）
- 章节概览：卡片网格展示 9 大章节，悬停上浮

### 章节详情页
- 面包屑导航
- 章节头部：图标 + 标题 + 描述
- Tab 切换展示各知识点，含核心公式
- 上下章节导航 + 工具/练习入口

### 化工计算器工具
- 两栏布局：左侧输入参数，右侧结果展示
- 输入框圆角、边框温润

### 自测练习
- 进度指引 + 选项卡片 + 即时反馈

## 动效与交互

- 悬停卡片轻微上浮（`translateY(-4px)`）+ 阴影加深
- 页面滚动平滑（`scroll-behavior: smooth`）
- 章节切换使用淡入 / 颜色过渡
- 按钮点击 `active:scale-95` 微反馈

## 设计禁忌

- 不使用冷色科技蓝或蓝紫色渐变
- 避免高饱和度霓虹色
- 避免过于简洁冰冷的"白墙"风格
- 不滥用动效影响阅读
- 不出现纯黑（`#000`），最深色为 `#2c2a29`
